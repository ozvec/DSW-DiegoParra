Objetivo del sitio

El presente OVI tiene como objetivo principal ilustrar el paso a paso y el correcto uso de las herramientas y elementos para el diseño, creación y maquetación de sitios web.

Objetivos específicos

  • objetivo específico 1:Complementar los conocimientos necesarios acerca de Visual Studio Code con el fin de aprender a usar el lenguaje de marcado HTML.
  • objetivo específico 2:Aprender y ccmpartir los métodos a través de los cuales podremos insertar y hacer uso de los Hipervínculos entre páginas web.
  • objetivo específico 3:Diseñar y maquetar paginas web, con ayuda de software y aplicativos pero con criterio propio, ejercitando nuestra creatividad y gestionando correctamente los espacios de las arquitecturas de las paginas Web.
