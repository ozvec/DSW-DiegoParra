Estructura básica de html5 (etiquetas, atributos y valores)

Objetivo del sitio

El presente OVI tiene como objetivo principal ilustrar el paso a paso y el correcto uso de las herramientas y elementos para el diseño, creación y maquetación de sitios web.

Objetivos específicos

  • Complementar los conocimientos necesarios acerca de Visual Studio Code con el fin de aprender a usar el lenguaje de marcado HTML.
  • Aprender y ccmpartir los métodos a través de los cuales podremos insertar y hacer uso de los Hipervínculos entre páginas web.
  • Diseñar y maquetar paginas web, con ayuda de software y aplicativos pero con criterio propio, ejercitando nuestra creatividad y gestionando correctamente los espacios de las arquitecturas de las paginas Web.

Contenido

  • ETIQUETAS: Las etiquetas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores.
  • ATRIBUTOS: Son aquellas características adicionales que permiten dar una configuración a un elemento o ajustan su comportamiento de diversas formas para cumplir los criterios del usuario.
  • VALORES: Son aquellas propiedades representadas en códigos numericos o caracteres especiales que definen particularidades de los elementos.
